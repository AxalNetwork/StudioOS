# 📬 3. Domain Deliverability Setup (CRITICAL)

Without this → your emails go to spam.

## 🛡️ SPF Record

Add to your DNS:

```bash
v=spf1 include:_spf.google.com ~all
```

## 🔐 DKIM (VERY IMPORTANT)

In Google Admin:

* Apps → Google Workspace → Gmail
* Go to:

  * “Authenticate Email”

Generate DKIM key → add DNS record

Then:
👉 Click **Start Authentication**

## 🧾 DMARC (Trust Layer)

Add DNS:

```bash
v=DMARC1; p=quarantine; rua=mailto:admin@axal.vc
```

Later you can upgrade to:

```bash
p=reject
```
